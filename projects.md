## Closette  
**Wardrobe & Outfit Management Application**  
**Tech:** Flutter, Dart, Firebase, REST API Integration  
**Status:** Live on Play Store

A complete wardrobe management solution enabling users to categorize clothing items, create outfits, and plan daily dressing routines. Designed with a focus on intuitive navigation and user experience.  
- Organized storage structure with Firebase
- User-friendly UI with responsive layout
- API-based operations for content storage
- Real-time updates for item management

**Link:** https://play.google.com/store/apps/details?id=my.app.new_closet_app

---

## Vegi  
**Smart Grocery Inventory Management**  
**Tech:** Flutter, Firebase Realtime Database, Provider State Management  
**Status:** Open Source (GitHub)

A grocery handling and management system that helps users track inventory, search products, and manage cart functionality.  
- Realtime data sync using Firebase Realtime DB  
- Provider for controlled state flow  
- Material Design UI with simple accessibility structure

**GitHub:** https://github.com/sangeeta2701/vegi

**E-Commerce & Cart Management**  
**Tech:** Flutter, Provider
